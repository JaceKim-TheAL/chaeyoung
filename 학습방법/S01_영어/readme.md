# 시험 리뷰

- 공부란 내가 모르는 것을 하나씩 알아 나가는 것
  - 몰랐던 문제
  - 틀렸던 문제
  - 햇갈린 문제 : 아리까리 했던걸 찍어서 푼것
  - 실수한 문제 : 아는건데 문제를 잘못 읽거나, 착각한 것!
- 똑같은 실수는 하지 말자
  - 오답 노트가 중요
  - 약한 과목일수록 복습에 충실
- 나의 강점과 약점을 파악 
  - 유형을 분석하고 파악하자
  - 어느곳에 먼저 집중해야할지 알수있다.
  - 아무리 복잡하고 어려운 문제도 하나씩 풀다보면 어느순간 해결된다.

<img src="../../img/문제해결방법.png"> 

- 문제의 유형
  - 문맥 : 문장의 맥락을 이해
  - 의미 : 문장이나 어휘의 의미를 이해하거나 올바른 해석
  - 문법 : 문장을 만드는 기법
- 독해
  - 긴문장과 연관된 문제들(3~4문제)
  - 문장에서 주어와 동사를 빠르게 캐치
  - 특히 메인문장의 동사를 찾으면 바로앞이 주어
