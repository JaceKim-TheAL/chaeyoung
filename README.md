# 채영이와 함께하는 노트

### 아빠가 하고싶은 말

- 세상에서 가장 사랑하는 채영아!
- 좋은말도 떄론 듣기싫고 꼰대처럼 보인다는것도 알지만 아빠는 늘 걱정하는 마음으로 고민하고 있어.
- 살아보니 하나님께서 만든 세상이 늘 공평하지만도 아름다운것만도 아니더라.
- 속물같이 들릴수도 있겠지만, 좀 더 현명하게 살아갔으면 하는 바램으로 한마디 할까해!
